# PHP Auto-Installer & Simple Switcher (Simple v1-B)

Script Bash minimalis untuk menginstal PHP FPM dan PHP CLI di server Debian/Ubuntu secara instan.

## 🚀 Fitur Utama
* **Fast Install** - Instalasi PHP versi standar secara cepat beserta paket-paket modul fundamental.
* **Basic Version Switcher** - Membantu beralih versi default PHP CLI dengan command `update-alternatives` terintegrasi.
* **FPM Status Dashboard** - Menampilkan status runtime service php-fpm secara langsung.

## 📋 Prasyarat
- **Sistem Operasi**: Ubuntu / Debian.
- **Hak Akses**: Pengguna harus memiliki akses `root` atau `sudo`.

## 🛠️ Cara Penggunaan

Anda dapat menjalankan script ini langsung menggunakan Main Launcher `main.sh` di root directory, atau secara standalone:

```bash
chmod +x php-installer.sh
sudo ./php-installer.sh
```
